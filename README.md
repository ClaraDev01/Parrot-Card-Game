# 🦜 Parrot Card Game

A themed memory card game featuring dancing parrots! Built with JavaScript, HTML and CSS — no libraries used!

## ✨ Features

- 🃏 Choose how many cards to play with (4 to 14, even numbers only)
- 🔀 Cards are randomly shuffled every game
- 🔄 3D flip animation when clicking a card
- ✅ Matched pairs stay face up permanently
- ⏱️ Wrong pairs flip back after 1 second
- 🏆 Win alert showing total moves count
- 📱 Fully responsive — works on mobile and desktop

## 🎯 How to Play

1. Enter the number of cards you want to play with (4, 6, 8, 10, 12 or 14)
2. Click a card to flip it
3. Try to find the matching pair
4. Match all pairs to win!

## 💻 How to Run Locally
```
git clone https://github.com/ClaraDev01/Parrot-Card-Game.git
```
Then open `index.html` in your browser — no installation needed!
